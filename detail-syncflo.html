개요

프로젝트 소개

SyncFlo는 기업의 구매·정산 업무를 하나의 플랫폼에서 통합 관리할 수 있는 B2B SaaS 서비스입니다. 기존에 엑셀과 이메일로 처리하던 복잡한 구매 프로세스를 디지털화하고, 담당자 간 실시간 협업이 가능하도록 설계했습니다.

웹 대시보드와 모바일 앱을 동시에 설계하며, 다양한 사용자 역할(구매 담당자, 승인자, 회계팀)의 워크플로우를 하나의 일관된 경험으로 통합하는 것이 핵심 과제였습니다.

01 — 문제 정의

사용자의 페인포인트를 분석했어요

기업 구매 담당자와의 심층 인터뷰를 통해 현장의 핵심 불편함을 발굴했습니다. 대부분의 기업이 여전히 아날로그 방식에 의존하고 있었고, 디지털 전환의 니즈는 높지만 기존 솔루션의 복잡성 때문에 도입을 꺼리고 있었습니다.

😩 문제
분산된 구매 정보,
파악이 어렵다
구매 요청, 승인 내역, 정산 현황이 이메일·엑셀·ERP에 분산되어 있어 현재 상태를 파악하는 데만 수시간이 걸립니다.
✦ 해결
통합 대시보드로
한눈에 파악
모든 구매 정보를 하나의 대시보드에 통합하고, 상태별 필터와 실시간 알림으로 즉각적인 현황 파악이 가능하도록 설계했습니다.
😩 문제
승인 프로세스가
느리고 불투명하다
결재 라인이 복잡하고, 승인 진행 상황을 실시간으로 확인할 수 없어 불필요한 커뮤니케이션이 반복됩니다.
✦ 해결
투명한 결재 흐름과
모바일 빠른 승인
결재 현황을 타임라인으로 시각화하고, 모바일 앱에서 언제 어디서든 승인·반려 처리가 가능한 플로우를 구현했습니다.
😩 문제
도입 장벽이 높고
학습 비용이 크다
기존 ERP 솔루션은 기능이 방대해 온보딩에 수 주일이 걸리고, 비IT 직군 직원들의 거부감이 큽니다.
✦ 해결
역할별 최적화된
간결한 인터페이스
사용자 역할(구매 담당자·승인자·회계팀)에 따라 필요한 기능만 보여주는 컨텍스추얼 UI로 학습 비용을 최소화했습니다.
02 — 목표, 방향성 설정

제품의 목표와 방향성을 설정했어요

구매 담당자와 회계팀이 주로 사용하는 웹 대시보드는 대량의 데이터를 효율적으로 탐색하고 처리하는 데 초점을 맞췄습니다. 복잡한 테이블 뷰와 분석 차트를 명확한 정보 계층으로 정리하고, 키보드 단축키와 일괄 처리 기능으로 파워 유저의 업무 효율을 높였습니다.

05 — 모바일 디자인

언제 어디서든, 그리고 일관된 경험

승인자(임원·팀장)를 위한 모바일 앱은 빠른 결재에 최적화되어 있습니다. 알림을 받는 즉시 요청 내역을 확인하고 한 탭으로 승인·반려가 가능합니다.

웹과 모바일을 아우르는 통합 디자인 시스템을 구축하여 어떤 플랫폼에서도 동일한 브랜드 경험을 제공하고, 향후 기능 확장 시에도 일관성을 유지할 수 있도록 했습니다.

📱
모바일 우선 알림 플로우
푸시 알림 → 상세 화면 진입 → 승인/반려까지 3탭 이내로 완료. 이동 중에도 업무 처리가 가능한 초간결 플로우를 설계했습니다.
🎨
컴포넌트 기반 디자인 시스템
버튼, 폼, 테이블, 차트 등 80+ 컴포넌트를 정의하고 라이트/다크 모드를 모두 지원하는 토큰 체계를 구축했습니다.
♿
접근성 WCAG AA 준수
색상 대비, 포커스 인디케이터, 스크린 리더 지원 등 기업 서비스에 요구되는 접근성 기준을 충족했습니다.
